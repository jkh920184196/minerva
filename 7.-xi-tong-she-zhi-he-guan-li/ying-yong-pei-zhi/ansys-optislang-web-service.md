# Ansys optiSLang Web Service

Ansys optiSLang Web Service 是一個平台，用於發布、管理和監控多個 optiSLang 專案。它使您能夠定義工作流程，自動化和標準化在 optiSLang 中執行的任務，並遠程執行這些工作流程。

使用 optiSLang Web Service 與 Ansys Minerva，用戶可以使用 optiSLang web 應用程序將項目提交到 optiSLang Web Service。必須在 Ansys Minerva 中啟用這些應用程序，以便用戶可以訪問它們。

使用 Ansys Minerva 與 optiSLang Web Service 提供了額外的好處，例如存儲和版本控制輸出文件、訪問作業歷史記錄以及搜索特定數據等。

提供的說明描述了如何使用 OAuth 集成 Ansys Minerva 和 optiSLang，允許 Ansys Minerva 實例充當 optiSLang Web Service 的身份提供者 (IDP)。
